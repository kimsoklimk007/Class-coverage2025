## 🚀 របៀបប្រើប្រាស់

### 1. Clone Repository
```bash
git clone <repository-url>
cd cambodian-school-management
```

### 2. បើកលើ Browser
```bash
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```

### 3. ឬប្រើប្រាស់ Live Server
```bash
# ប្រើ VSCode Live Server Extension
# ឬ Python
python -m http.server 8000
# បន្ទាប់មក ចូល http://localhost:8000
```
